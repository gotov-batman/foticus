# foticus (редактор фотографий)
## это приложение созданно для:
- Редактировать качество фотографий
- Разворачивать фотографий
- Обесцвечивать фотографии
- Сохранять фотографий
## Использует:
- код **python**
- модуль **PyQt5**
- модуль **os**
- модуль **PIL**
### Требования:
- pentium 2
- монитор
- встроенная графика
- 256 mb оперативной памяти
